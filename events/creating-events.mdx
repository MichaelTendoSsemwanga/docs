## Prerequisites

Before creating your first event, ensure you have:

### Account Requirements
- An active Loft organization account
- **Required Role**:
  - Administrator
  - Event Manager
  - Or custom role with event creation permissions

### Information to Prepare
1. **Event Details**
   - Event name and description
   - Date(s) and time(s)
   - Location details (physical/virtual)
   - Expected number of attendees

2. **Media Assets**
   - High-quality banner image (recommended: 1920x1080px)
   - Organization logo (square, min. 512x512px)
   - Venue photos or virtual event graphics

3. **Business Information**
   - Refund policy
   - Contact information
   - Tax settings (if applicable)

## Step 1: Event Creation

### 1.1 Accessing the Event Creation Wizard
1. Log in to your Loft account
2. From the dashboard, locate and click the **+ Create Event** button
3. Select your event type:

#### Event Type Options
| Type | Best For | Key Features |
|------|----------|-------------|
| **In-Person** | Physical events with venue | Venue management, capacity limits, floor plans |
| **Online** | Virtual events | Streaming integration, virtual lobbies, digital content |
| **Hybrid** | Combined physical/virtual | Both in-person and online components |

### 1.2 Initial Event Setup
1. **Basic Information**
   - **Event Name**: Clear, descriptive title (max 100 characters)
     - Example: `Annual Tech Conference 2024`
   - **Short Description**: Brief summary (max 160 characters for SEO)
   - **Full Description**: Detailed event information (supports Markdown)
   - **Category & Tags**: Select relevant categories and add tags for discoverability

2. **Date & Time Configuration**
   - **Start/End Date & Time**: Set in your organization's timezone
   - **Multiple Sessions**: Toggle on for multi-day or multi-session events
   - **Time Zone**: Automatically set to organization default (can be overridden)

## Step 2: Event Configuration

### 2.1 Location Setup

#### For In-Person Events
1. **Venue Details**
   - Venue name and address
   - Room/space name (if applicable)
   - Parking and transportation information

2. **Venue Capacity**
   - Set maximum capacity
   - Configure room layouts
   - Add accessibility information

#### For Online Events
1. **Virtual Event Platform**
   - Select from integrated platforms (Zoom, Vimeo, etc.)
   - Configure streaming settings
   - Set up virtual waiting rooms

2. **Technical Requirements**
   - Specify required software/plugins
   - Add system requirements
   - Include test session information

### 2.2 Event Schedule
1. **Session Management**
   - Add individual sessions/tracks
   - Assign speakers and rooms
   - Set capacity for each session

2. **Agenda Builder**
   - Drag-and-drop interface
   - Add breaks and networking sessions
   - Include session descriptions and materials

## Step 3: Ticket Setup

### 3.2 Pricing Strategies
1. **Tiered Pricing**

2. **Discount Codes**
   - Percentage or fixed amount off
   - Usage limits
   - Expiration dates

## Troubleshooting

### Common Issues & Solutions

#### Registration Problems
| Issue | Possible Cause | Solution |
|-------|----------------|-----------|
| Payment declined | Card issues | Try different payment method |
| Ticket not showing | Sold out or inactive | Check ticket settings |
| Form not submitting | Validation errors | Check required fields |

#### Technical Issues
- **Page not loading**: Clear cache or try incognito mode
- **Upload failures**: Check file size and format
- **Email delivery issues**: Verify SPF/DKIM records
