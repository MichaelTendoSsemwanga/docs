# Virtual Events with Loft

*Create immersive online experiences with Loft's virtual event platform. This guide covers everything from initial setup to post-event analytics for your virtual conferences, webinars, and hybrid events.*

## Quick Start

### Accessing Virtual Events
1. Log in to your Loft account
2. Click **Create Event** > **Virtual**
3. Follow the setup wizard

### Key Features
- **Virtual Venues**: Customizable event spaces
- **Live Streaming**: Built-in and third-party options
- **Engagement Tools**: Q&A, polls, networking
- **Analytics**: Real-time and post-event metrics

## Step 1: Event Setup

### 1.1 Basic Information
1. **Event Details**
   - Event name and description
   - Date and time (with timezone)
   - Event category and tags
   - Custom URL

2. **Virtual Venue**
   - Select a template:
     - Conference (multiple sessions)
     - Webinar (single presentation)
     - Expo (virtual booths)
     - Custom (build from scratch)

## Step 2: Configure Event Settings

### 2.1 Access Control
1. **Registration**
   - Public/Private event
   - Approval workflow
   - Custom registration form

2. **Security**
   - Waiting room
   - Moderator controls
   - Attendee permissions

### 2.2 Branding
1. Upload assets:
   - Logo (PNG, 200x50px)
   - Banner (1920x1080px)
   - Color scheme (HEX codes)
   - Custom CSS (Pro plan+)

## Step 3: Session Management

### 3.1 Create Sessions
1. Click **Sessions** > **+ New Session**
2. Configure:
   - Title and description
   - Date/time
   - Session type (live, pre-recorded, on-demand)
   - Speakers and moderators

### 3.2 Session Types
- **Main Stage**: Keynotes and panels
- **Breakout Rooms**: Small group discussions
- **Networking**: 1:1 or group video chats
- **Expo Hall**: Virtual booths and sponsors

## Step 4: Engagement Tools

### 4.1 Interactive Features
1. **Live Q&A**
   - Upvote questions
   - Moderator approval
   - Anonymous submission

2. **Polls & Surveys**
   - Multiple choice
   - Word cloud
   - Quiz mode

3. **Networking**
   - AI-powered matchmaking
   - Scheduled 1:1 meetings
   - Group video chats

## Step 5: Go Live

### 5.1 Technical Check
1. **Internet Speed**
   - Minimum: 5Mbps upload
   - Recommended: 10Mbps+ upload
   - Test at [speedtest.net](https://www.speedtest.net/)

2. **Hardware**
   - Webcam (1080p recommended)
   - Microphone (USB mic preferred)
   - Lighting (front-facing)

### 5.2 Start Your Event
1. **30 Minutes Before**
   - Test audio/video
   - Load presentations
   - Brief speakers

2. **Go Live**
   - Open virtual doors
   - Welcome attendees
   - Start with housekeeping

## Troubleshooting

### Common Issues

#### Connection Problems
1. **Host**
   - Use wired connection
   - Close bandwidth-heavy apps
   - Lower video quality

2. **Attendees**
   - Browser requirements
   - Clear cache/cookies
   - Try incognito mode
