# Virtual & Hybrid Events with Loft

*Deliver exceptional online experiences that engage and inspire. This guide covers everything you need to plan, execute, and measure successful virtual and hybrid events on the Loft platform.*

## Quick Start

### Before You Begin
1. **Account Access**
   - Log in to [Loft Dashboard](https://app.loft.events)
   - Navigate to **Events** > **Create New**
   - Select **Virtual** or **Hybrid** event type

2. **System Requirements**
   - **Organizer**:
     - Modern web browser (Chrome, Firefox, Edge)
     - Webcam & microphone
     - Stable internet (5Mbps+ upload)
   - **Attendees**:
     - Any device with internet access
     - No software downloads required

### Key Features at a Glance

| Feature | Virtual Events | Hybrid Events |
|---------|---------------|---------------|
| **Live Streaming** | ✓ Built-in RTMP | ✓ Multi-camera support |
| **Virtual Venues** | ✓ Customizable | ✓ Branded spaces |
| **Networking** | ✓ 1:1 & group | ✓ Cross-platform |
| **Engagement** | ✓ Polls, Q&A | ✓ Live translation |
| **Analytics** | ✓ Real-time | ✓ Comprehensive |
| **On-demand** | ✓ Automatic | ✓ Session-based |

> **Pro Tip**: Bookmark the [Virtual Events Dashboard](https://app.loft.events/virtual) for quick access to all your virtual events.

## Step 1: Event Setup

### 1.1 Event Details
1. **Basic Information**
   - **Event Name**: Clear, descriptive title (max 100 chars)
   - **Description**: Rich text editor with media support
   - **Date & Time**: 
     - Start/end dates
     - Timezone selector
     - Schedule in multiple timezones

2. **Event Type**
   - **Virtual**: Online-only experience
   - **Hybrid**: Combines in-person and virtual elements
   - **Webinar**: Single presenter to many attendees
   - **Conference**: Multiple tracks and sessions

3. **Virtual Venue Selection**
   | Venue Type | Best For | Capacity | Features |
   |------------|----------|----------|----------|
   | **Theater** | Keynotes | 10,000+ | Stage, backstage, Q&A |

   | **Networking** | Social events | 100-1,000 | Video tables, 1:1 meetings |
   | **Custom** | Unique needs | Varies | Fully customizable |

## Step 5: Going Live & Event Execution

### 5.1 Pre-Event Testing
1. **Technical Requirements**
   - **Internet Connection**:
     - Minimum: 5Mbps upload/download
     - Recommended: 10Mbps+ upload/download
     - Wired connection preferred
     - Test at [speedtest.net](https://www.speedtest.net/)
   - **Hardware Setup**:
     - Webcam (1080p recommended)
     - External microphone (USB or XLR)
     - Headphones (to prevent echo)
     - Ring light or soft front lighting

2. **Run-Through**
   - **1 Week Before**:
     - Full technical rehearsal
     - Speaker dry runs
     - Backup presenter briefings
   - **Day Before**:
     - Final AV check
     - Internet speed test
     - Backup equipment test

### 5.2 Event Day Execution
1. **Pre-Show (60 min before)**
   - Open virtual venue
   - Test all streaming connections
   - Verify speaker and moderator access
   - Start background music/logo loop

2. **Countdown (30 min before)**
   - Begin attendee check-in
   - Start welcome slides/video
   - Monitor attendee engagement
   - Address technical issues

3. **Showtime (Go Live)**
   - Start recording
   - Begin live stream
   - Welcome message
   - Housekeeping announcements
   - Introduce first session

4. **During the Event**
   - Monitor engagement metrics
   - Moderate Q&A and chats
   - Launch scheduled activities
   - Address technical issues

5. **Session Transitions**
   - Clear session end announcements
   - Next session countdown
   - Breakout room management
   - Speaker handoffs

6. **Event Conclusion**
   - Thank attendees
   - Share next steps
   - Post-event survey
   - On-demand access information

## Troubleshooting & Support

### Common Issues & Solutions

| Issue | Quick Fix | When to Escalate |
|-------|-----------|------------------|
| **Poor Video Quality** | Switch to wired connection, lower resolution | Persistent issues after troubleshooting |
| **Audio Echo** | Use headphones, mute when not speaking | Echo persists with headphones |
| **Connection Drops** | Restart router, use mobile hotspot | Frequent disconnections |
| **Login Issues** | Reset password, clear cache | Account locked or compromised |
| **Feature Not Working** | Check browser compatibility, update app | Feature confirmed to be down |
