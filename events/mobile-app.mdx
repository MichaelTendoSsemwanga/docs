# Event Mobile App Setup Guide

*Create a professional mobile app for your event in minutes with Loft's intuitive app builder. This guide walks you through every step from initial setup to app store submission and attendee engagement.*

## Getting Started

### Accessing the App Builder
1. Log in to your Loft account
2. Go to **Events** > **Your Event** > **Mobile App**
3. Click **Create App** to begin

### Before You Begin
- Have your event details ready (schedule, speakers, sponsors)
- Prepare your logo and brand colors
- Gather high-quality images (recommended size: 1024x500px for headers)
- Know your target app features

## Step 1: Configure Basic Settings

### 1.1 App Information
1. **App Name**
   - Enter your event name (max 30 characters for app stores)
   - Add a short description (50 characters)
   - Full description (4000 characters max)

2. **App Type Selection**
   - **Native App**: Full app store distribution
     - Requires developer accounts
     - Best for large events
   - **Web App**: Instant access via browser
     - No app store needed
     - Quick setup

### 1.2 App Store Setup (Native Only)
1. **Apple App Store**
   - Apple Developer Account ($99/year)
   - App Store Connect setup
   - Privacy policy URL required

2. **Google Play Store**
   - Google Play Developer Account ($25 one-time)
   - Store listing details
   - Content rating questionnaire

## Step 2: Customize Your App

### 2.1 Branding
1. **Logo & Icons**
   - Upload app icon (1024x1024px, PNG)
   - Set splash screen (1242x2436px)
   - Choose tab bar icons

2. **Color Scheme**
   - Primary & secondary colors
   - Background colors
   - Text colors (ensure contrast)
   - Action button colors

### 2.2 Navigation
1. **Menu Structure**
   - Drag-and-drop menu items
   - Reorder sections
   - Set home screen

2. **Tab Bar**
   - Choose 3-5 main tabs
   - Label clearly
   - Use standard icons

## Step 3: Add Event Content

### 3.1 Core Sections
1. **Schedule**
   - Import from your Loft event
   - Add sessions manually
   - Set tracks and categories

2. **Speakers**
   - Add speaker profiles
   - Include photos and bios
   - Link to sessions

3. **Sponsors**
   - Add sponsor logos
   - Create sponsor tiers
   - Link to sponsor websites

### 3.2 Additional Content
1. **Maps**
   - Upload venue maps
   - Set locations
   - Add custom pins

2. **Documents**
   - Upload PDFs
   - Add links
   - Organize in folders

## Step 4: Enable Features

### 4.1 Engagement Tools
1. **Networking**
   - Enable attendee profiles
   - Set up 1:1 messaging
   - Create meetup scheduling

2. **Live Features**
   - Push notifications
   - Live polling
   - Q&A sessions

### 4.2 Analytics
1. **Track Usage**
   - User engagement
   - Feature popularity
   - Session attendance

2. **Lead Retrieval**
   - Set up lead capture
   - Export leads
   - Track scans

## Step 5: Test & Launch

### 5.1 Testing
1. **Preview**
   - Use the Loft Preview app
   - Test on iOS and Android
   - Check all links and forms

2. **Beta Testing**
   - Invite testers
   - Collect feedback
   - Make final adjustments

### 5.2 App Store Submission
1. **Prepare Assets**
   - Screenshots (multiple sizes)
   - App store descriptions
   - Privacy policy

2. **Submit for Review**
   - Apple: 24-48 hour review
   - Google: 1-3 day review
   - Monitor status in dashboard

## Step 6: Promote Your App

### 6.1 Pre-Event
1. **Early Access**
   - Email campaign
   - Social media teasers
   - Download incentives

2. **On-Site**
   - QR codes on signage
   - Download stations
   - Emcee announcements

### 6.2 During Event
1. **Gamification**
   - Points for app actions
   - Leaderboard
   - Rewards

2. **Updates**
   - Push notifications
   - Schedule changes
   - Important announcements

## Troubleshooting

### Common Issues

#### App Not Updating
1. **Check Sync Status**
   - Go to **App** > **Settings** > **Sync**
   - Force refresh content
   - Check internet connection

2. **Update App**
   - Visit app store
   - Check for updates
   - Reinstall if needed

#### Login Problems
1. **Reset Password**
   - Use "Forgot Password"
   - Check spam folder
   - Try social login

2. **Account Issues**
   - Verify email
   - Check event registration
   - Contact support

## Step 2: Set Up App Structure

### 2.1 Create Your App
1. Go to **Event Dashboard** > **Mobile App**
2. Click **Create New App**
3. Enter basic info:
   - App name
   - Event selection
   - Primary language
   - Timezone

### 2.2 Navigation Setup
1. **Main Menu**:
   - Schedule
   - Speakers
   - Attendees
   - Map
   - Notifications
   - More

2. **Custom Tabs** (up to 5):
   - Sponsors
   - Surveys
   - Gamification
   - Resources
   - Social Feed

## Step 3: Customize Design

### 3.1 Branding
1. **Logo & App Icon**
   - Upload high-res logo (min 1024x1024px)
   - Generate app icon (auto-resized)
   - Set splash screen

2. **Color Scheme**
   - Primary color
   - Secondary color
   - Background colors
   - Text colors

### 3.2 UI Customization
- **Fonts**: Choose from Google Fonts

- **Layout**: Card/list view options

## Troubleshooting

### Common Issues & Solutions

#### App Installation
1. **Problem**: App not in store
   - Check if approved
   - Verify country/region
   - Try direct link

2. **Problem**: Login failures
   - Reset password
   - Check email verification
   - Try web version

#### Functionality
1. **Problem**: Content not loading
   - Check internet
   - Update app
   - Clear cache

2. **Problem**: Push notifications
   - Enable in device settings
   - Check app permissions
   - Reinstall if needed
