# Loft Event Mobile App Guide

*Create a powerful, branded mobile experience for your event with Loft's intuitive app builder. This comprehensive guide covers everything from initial setup to advanced features and attendee engagement.*

## Quick Start

### System Requirements
- **Loft Account**: Active subscription with app builder access
- **Images**: 
  - Logo: 1024x1024px (PNG, transparent background)
  - Header: 1920x1080px (JPG/PNG)
  - Speaker/Sponsor: 800x800px (JPG/PNG)
- **Content**: 
  - Event schedule (sessions, speakers, sponsors)
  - Venue information
  - Social media handles

### Accessing the App Builder
1. Log in to your Loft account at [app.loft.events](https://app.loft.events)
2. Navigate to **Events** > **Your Event** > **Mobile App**
3. Click **Create App** or **Edit App** to begin

> **Pro Tip**: Bookmark the App Builder for quick access: [https://app.loft.events/events/your-event/app](https://app.loft.events/events/your-event/app)

## Step 1: Configure App Settings

### 1.1 Basic Information
1. **App Name & Details**
   - **Display Name**: Max 30 characters (e.g., "TechConf 2024")
   - **Short Description**: 50 characters for app stores
   - **Full Description**: 4000 characters max (SEO optimized)
   - **Keywords**: Comma-separated for app store optimization

2. **App Type Selection**
   - **Native App** (Recommended)
     - **Pros**: Full app store presence, push notifications, offline access
     - **Requirements**: 
       - Apple Developer Account ($99/year)
       - Google Play Developer Account ($25 one-time)
   - **Progressive Web App (PWA)**
     - **Pros**: No app store needed, instant updates
     - **Best for**: Smaller events, quick deployment
   - **Hybrid** (Native + PWA)
     - **Benefits**: Maximum reach and functionality

### 1.2 App Store Configuration

#### Apple App Store
1. **Prerequisites**
   - Active Apple Developer Program membership
   - App Store Connect access
   - Privacy policy URL (required)

2. **Setup Process**
   ```
   App Store Connect > My Apps > + (New App)
   ```
   - Platform: iOS
   - Bundle ID: com.yourcompany.yourevent
   - SKU: YYYYMMDD-EVENT-CODE
   - User Access: Full/App Manager

#### Google Play Store
1. **Prerequisites**
   - One-time $25 developer registration
   - Google Play Developer account
   - Privacy policy URL
   - App content rating

2. **Setup Process**
   ```
   Play Console > All Applications > Create Application
   ```
   - Default Language: English
   - App Name: Your Event Name
   - App Type: Application
   - Free/Paid: Select as applicable

## Step 2: Design & Branding

### 2.1 Brand Assets
1. **App Icons**
   - **Format**: PNG with transparent background
   - **Sizes**:
     - iOS: 1024x1024px (App Store)
     - Android: 512x512px (Play Store)
     - Web: 192x192px (PWA)
   - **Best Practices**:
     - Keep important content centered
     - Avoid thin lines/small text
     - Test on light/dark backgrounds

2. **Splash Screens**
   - **Dimensions**: 1242x2688px (iPhone), 1536x2048px (iPad)
   - **Loading Indicator**: Customizable spinner color
   - **Background**: Solid color or subtle gradient

### 2.2 Color Scheme & Typography

#### Color Palette
1. **Primary Colors**
   - Brand color (used for main actions)
   - Secondary color (accent)
   - Background colors (light/dark mode)

2. **Accessibility**
   - Minimum contrast ratio: 4.5:1 (WCAG AA)
   - Test color combinations:
     - Primary text on white
     - White text on primary
     - Secondary text on backgrounds

#### Typography
1. **Font Selection**
   - **System Fonts**: Default for performance
   - **Custom Fonts**: Upload .ttf or .otf files
   - **Recommended**:
     - Headings: Bold, easy-to-read sans-serif
     - Body: Clean, legible at small sizes

2. **Text Hierarchy**
   - H1: 24-32px
   - H2: 20-24px
   - Body: 16-18px
   - Captions: 12-14px

### 2.3 UI Components

#### Navigation
1. **Tab Bar**
   - Max 5 items
   - Standard icons:
     - Home
     - Schedule
     - Speakers
     - Map
     - More

2. **Menu Structure**
   ```
   - Home
   - Schedule
     - Day 1
     - Day 2
   - Speakers
   - Sponsors
   - Venue
   - My Schedule
   - Notifications
   - Profile
   ```

#### Interactive Elements
1. **Buttons**
   - Primary: Solid, high contrast
   - Secondary: Outlined
   - Size: Min 44x44px touch target

2. **Cards**
   - Consistent corner radius (8-12px)
   - Subtle shadow for depth
   - Clear visual hierarchy

## Step 3: Content Management

### 3.1 Schedule & Sessions

#### Importing Schedule
1. **From Loft Event**
   - Go to **Content** > **Schedule**
   - Click **Import from Event**
   - Map session fields:
     - Title
     - Date/Time
     - Location
     - Tracks
     - Speakers

2. **Manual Entry**
   - Click **+ Add Session**
   - Required fields:
     - Session title
     - Start/end time
     - Location
     - Session type
   - Optional:
     - Session description
     - Speaker assignments
     - Resources/attachments

#### Session Types

2. **Special Sessions**
