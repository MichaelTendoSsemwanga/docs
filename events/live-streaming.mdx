## Troubleshooting

### Common Issues
| Issue | Solution |
|-------|----------|
| Lag/buffering | Lower bitrate, check connection |
| Audio sync | Adjust delay, restart encoder |
| Dropped frames | Reduce resolution, check CPU |
| No signal | Check cables, restart devices |
