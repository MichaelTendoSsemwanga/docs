## Troubleshooting

### Common Issues
| Issue | Solution |
|-------|----------|
| Low Open Rates | Improve subject lines, clean list |
| High Bounce Rate | Verify emails, remove invalid addresses |
| Spam Folders | Check authentication, review content |
| Low Clicks | Test CTAs, improve content |
| Unsubscribes | Review frequency, segment better |
