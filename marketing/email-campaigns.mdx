# Email Marketing with Loft

*Create, automate, and optimize email campaigns that drive event success. This comprehensive guide covers everything from list building to advanced segmentation and analytics in the Loft platform.*

## Quick Start

### Accessing Email Marketing
1. Log in to your Loft account
2. Go to **Marketing** > **Email Campaigns**
3. Click **Create Campaign**

### Key Features
- Drag-and-drop email builder
- Advanced segmentation
- A/B testing
- Automated workflows
- Real-time analytics
- Pre-built templates

## Step 1: Create a New Campaign

### 1.1 Campaign Setup
1. Click **+ New Campaign**
2. Select campaign type:
   - **Regular Email**: One-time send
   - **Automated Series**: Drip campaigns
   - **A/B Test**: Compare versions
   - **Event Invitation**: With RSVP tracking

### 1.2 Basic Settings
1. **Campaign Details**
   - Campaign name (internal use)
   - Email subject line
   - Preview text
   - Sender name & email
   - Reply-to address

2. **Audience**
   - Select from existing lists
   - Create new segment
   - Exclude specific groups
   - Test group (10% of audience)

## Step 2: Design Your Email

2. **Customize Design**

### 2.2 Content Blocks

## Step 2: Choose Email Types

### 2.1 Core Email Sequence
1. **Save the Date** (6-8 weeks before)
2. **Early Bird Announcement** (4-6 weeks)
3. **Speaker/Agenda Reveal** (3-4 weeks)
4. **Last Chance Reminder** (1 week)
5. **Final Details** (1-2 days before)
6. **Post-Event Thank You** (24-48 hours after)

### 2.2 Automated Emails
- **Welcome Series**: For new subscribers
- **Abandoned Registration**: For incomplete signups
- **Session Reminders**: 1 hour before each session
- **Re-engagement**: For inactive subscribers

## Step 3: Design Emails

### 3.1 Email Builder Basics
1. Select a template from **Marketing** > **Emails**
2. Use drag-and-drop editor
3. Add sections:
   - Header with logo
   - Hero image
   - Main content
   - CTA buttons
   - Footer

### 3.2 Design Best Practices
- **Mobile-First**: 600px width max
- **Images**: Optimize size, add alt text
- **Buttons**: Minimum 44x44px, high contrast
- **Whitespace**: 15-20px between sections
- **Fonts**: Web-safe, minimum 14px

## Step 4: Write Compelling Copy

### 4.1 Subject Line Formula
```
[Benefit] + [Urgency/Scarcity] + [Personalization]
```
**Example**: "Your Exclusive Pass: 24-Hour Flash Sale, [First Name]!"

### 4.2 Email Copy Structure
1. **Greeting**: Personalized when possible
2. **Opening Hook**: Address pain point or desire
3. **Value Proposition**: What's in it for them
4. **Social Proof**: Testimonials or stats
5. **Clear CTA**: One primary action
6. **PS**: Add urgency or bonus

## Step 5: Automate Campaigns

### 5.1 Create a Drip Campaign
1. Go to **Automation** > **Campaigns**
2. Choose trigger (e.g., "User registers")
3. Add emails to sequence
4. Set delays between emails
5. Add conditions (e.g., "Only if attended session")

### 5.2 Behavioral Triggers
- Email opens/clicks
- Website visits
- Session attendance
- Form submissions
- Purchase activity

## Step 6: Test & Send

### 6.1 Pre-Send Checklist
- [ ] Test subject lines (use A/B testing)
- [ ] Check all links
- [ ] Preview on mobile/desktop
- [ ] Test spam score (aim for < 3/10)
- [ ] Set up tracking (UTM parameters)

### 6.2 Best Times to Send
- **B2B**: Tuesday-Thursday, 9-11 AM
- **B2C**: Weekends, 7-9 PM
- **Global Audiences**: Segment by timezone
- **Test**: Your audience might be different

## Step 7: Analyze Results

### 7.1 Key Metrics to Track
| Metric | Goal | How to Improve |
|--------|------|----------------|
| Open Rate | 20-30% | Better subject lines |
| Click Rate | 2-5% | Clearer CTAs |
| Unsubscribes | <0.5% | Better targeting |
| Bounce Rate | <2% | Clean your list |
| Conversions | Varies | Optimize landing pages |

### 7.2 A/B Testing Ideas
1. **Subject Lines**: Question vs. statement
2. **Send Times**: Morning vs. evening
3. **From Name**: Person vs. company
4. **Content Length**: Short vs. detailed
5. **CTA Buttons**: Color, text, placement

## Pro Tips & Best Practices

### List Management
1. **Segmentation**
   - By interest/topic
   - Engagement level
   - Past attendance
   - Ticket type

2. **List Hygiene**
   - Remove hard bounces immediately
   - Re-engage inactive subscribers
   - Clean list quarterly

### Deliverability
1. **Authentication**
   - Set up SPF, DKIM, DMARC
   - Use dedicated IP if sending >50k/month
   - Monitor blacklists

2. **Engagement**
   - Send consistent, valuable content
   - Make unsubscribing easy
   - Monitor spam complaints

## Troubleshooting

### Common Issues & Solutions

#### Low Open Rates
1. **Problem**: Emails not reaching inbox
   - Check spam folder
   - Test deliverability
   - Review authentication

2. **Problem**: Poor subject lines
   - Use power words
   - Keep under 50 characters
   - Test emojis

#### Low Click-Through Rates
1. **Problem**: Unclear CTAs
   - Use action-oriented language
   - Make buttons prominent
   - Reduce competing links

2. **Problem**: Content mismatch
   - Align subject line with content
   - Deliver on promises
   - Improve content relevance
