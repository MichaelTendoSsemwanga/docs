# Check-in & Badge Management with Loft

*Streamline your event check-ins and badge management with Loft's powerful tools. This comprehensive guide covers everything from initial setup to on-site execution and post-event reporting.*

## Quick Start

### System Requirements
- **Mobile Devices**:
  - iOS 13.0+ (iPhone/iPad)
  - Android 8.0+
  - Recommended: 5.5"+ screen size
- **Printers**:
  - Supported models listed in Loft Help Center
  - Network connectivity (WiFi/Ethernet)

### Before You Begin
1. Download the Loft Check-in App
2. Set up your event in Loft
3. Prepare attendee data
4. Design your badges (or use templates)

## Step 1: Configure Check-in Settings

### 1.1 Event Setup
1. Log in to your Loft account
2. Go to **Events** > **Your Event** > **Check-in**
3. Configure settings:
   - **Check-in Windows**: Set early/late check-in times
   - **Multiple Check-ins**: Allow/restrict
   - **Required Fields**: Info to collect at check-in
   - **Custom Questions**: Add event-specific questions

### 1.2 Staff Access
1. **Add Team Members**
   - Enter email addresses
   - Assign roles (Admin, Manager, Staff)
   - Set permissions
2. **Download Check-in App**
   - iOS: App Store
   - Android: Google Play
   - Enterprise: MDM deployment available

## Step 2: Design & Print Badges

### 2.1 Badge Templates
1. **Choose Template**
   - Browse Loft's template library
   - Filter by event type/industry
   - Preview designs
2. **Customize Design**
   - Add your logo
   - Choose color scheme
   - Select badge size (standard: 3.5" x 2")

### 2.2 Badge Fields
1. **Standard Fields**
   - First/Last Name
   - Company
   - Job Title
   - QR Code (auto-generated)
2. **Custom Fields**
   - Session tracks
   - Meal preferences
   - Access levels

### 2.3 Printing Options
1. **On-Demand Printing**
   - Print as attendees check in
   - Ideal for smaller events
2. **Pre-Printing**
   - Bulk print before event
   - Requires complete attendee list
3. **Self-Service Kiosks**
   - Attendees print their own badges
   - Staff-assisted option available

## Step 3: Set Up Check-in Stations

### 3.1 Hardware Setup
1. **Devices**
   - Tablets/phones for check-in
   - Badge printers
   - Label printers (for wristbands)
   - Cash drawers (if needed)
2. **Connectivity**
   - WiFi network with strong signal
   - Backup mobile hotspots
   - Offline mode setup

### 3.2 Station Types
1. **Express Check-in**
   - QR code scanning
   - Fast processing
   - Minimal staff needed
2. **Full-Service**
   - Manual name lookup
   - Payment collection
   - Issue resolution
3. **VIP/Media**
   - Dedicated line
   - Special handling
   - Additional services

## Step 4: Manage Check-in Process

### 4.1 Check-in Methods
1. **QR Code Scan**
   - From email/mobile ticket
   - Printed ticket
   - Add to Wallet (iOS/Android)
2. **Manual Lookup**
   - Search by name/email
   - Registration reference
   - ID verification

### 4.2 Handling Scenarios
1. **No Registration**
   - On-site registration option
   - Payment collection
   - Badge printing
2. **Already Checked In**
   - View check-in time/location
   - Re-print badge if needed
   - Flag suspicious activity
3. **Special Cases**
   - VIPs
   - Speakers
   - Press/Media
   - Accessibility needs

## Step 5: Real-time Monitoring

### 5.1 Dashboard
1. **Live Stats**
   - Check-ins by hour
   - Queue lengths
   - Staff productivity
2. **Alerts**
   - Long wait times
   - Printer issues
   - System errors

### 5.2 Reports
1. **Attendance Reports**
   - Check-in times
   - Session attendance
   - No-shows
2. **Export Options**
   - CSV/Excel
   - PDF summaries
   - Custom date ranges

## Troubleshooting

### Common Issues

#### QR Code Won't Scan
1. Clean camera lens
2. Increase screen brightness
3. Move to better lighting
4. Try manual entry

#### Printer Not Working
1. Check power and connections
2. Verify paper/ribbon
3. Restart printer
4. Test with printer utility

#### Offline Mode Issues
1. Check sync status
2. Verify offline data downloaded
3. Restart app
4. Contact support if issues persist
