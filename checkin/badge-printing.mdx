## Step 1: Plan Your Badge Layout

### 1.1 Determine Badge Size
- **Standard Sizes**:
  - 3.5" x 2" (Credit card)
  - 4" x 3" (Standard name badge)
  - 8.5" x 11" (Full page)
- **Custom Sizes**: Available for special requirements

### 1.2 Essential Elements
1. **Required**:
   - First & last name
   - Company/organization
   - Badge type/access level
   - Barcode/QR code

2. **Optional**:
   - Headshot
   - Social media handles
   - Session schedule
   - Emergency contact

## Step 2: Design Your Badge

### 2.1 Access Design Tools
1. Go to **Event Dashboard** > **Check-in** > **Badge Designer**
2. Choose a template or start from scratch
3. Use drag-and-drop editor to customize

### 2.2 Design Elements
- **Branding**: Upload logos, set brand colors
- **Text Fields**: Drag to position, set font/size
- **Images**: Add headshots or sponsor logos
- **Barcodes**: Auto-generate from attendee data
- **Zones**: Define areas for different content types

### 2.3 Advanced Features
- **Conditional Formatting**: Different designs for ticket types
- **QR Codes**: Link to attendee profiles
- **Watermarks**: For special access levels
- **Backside Printing**: Additional info or sponsor messages

## Step 3: Set Up Printing

### 3.1 Choose Printing Method

| Method | Best For | Pros | Cons |
|--------|----------|------|------|
| **Pre-printed** | Large events | Professional finish, fast distribution | No last-minute updates |
| **On-demand** | Small/medium events | Real-time updates, no waste | Slower, requires equipment |
| **Hybrid** | Most events | Balance of speed and flexibility | More complex setup |

### 3.2 Printer Configuration
1. **Select Printer Type**:
   - Thermal (direct/transfer)
   - Inkjet
   - Laser

2. **Configure Settings**:
   - Paper size/type
   - Print quality
   - Color profile
   - Duplex settings

## Step 4: Test Printing

### 4.1 Print Test Batch
1. Select 5-10 sample attendees
2. Print one badge of each type
3. Check:
   - Print quality
   - Alignment
   - Color accuracy
   - Barcode scannability

### 4.2 Adjust as Needed
- Resize elements
- Adjust margins
- Update color profiles
- Test different paper stocks

## Step 5: Bulk Printing

### 5.1 Prepare for Bulk Print
1. Sort attendee list (alphabetical/by company)
2. Load sufficient supplies
   - Badge stock
   - Ink/toner
   - Lanyards

### 5.2 Start Print Job
1. Select all attendees or filter groups
2. Choose print queue order
3. Set batch size (recommended: 50-100)
4. Monitor progress
5. Handle jams/pauses as needed

## Step 6: Onsite Printing

### 6.1 Self-Service Kiosks
1. Set up kiosk stations
2. Provide clear instructions
3. Include:
   - Touchscreen monitor
   - Badge printer
   - Lanyard dispenser
   - Staff assistance button

### 6.2 Staffed Check-in
1. Train volunteers on:
   - Badge lookup
   - Reprint process
   - Issue resolution
2. Keep backup supplies nearby

## Pro Tips & Best Practices

### Design Tips
1. **Readability**:
   - Minimum 24pt font for names
   - High contrast colors
   - Clear hierarchy

2. **Functionality**:
   - Place barcodes in consistent location
   - Include emergency info on back
   - Use color coding for quick identification

### Printing Tips
1. **Before Event**:
   - Order 10% extra supplies
   - Create badge design cheat sheet
   - Test different badge holders

2. **During Event**:
   - Have a reprint station
   - Keep blank badges handy
   - Monitor ink/toner levels

## Troubleshooting

### Common Issues & Solutions

#### Printing Problems
| Issue | Likely Cause | Solution |
|-------|-------------|----------|
| Faded print | Low ink/toner | Replace cartridge |
| Misaligned | Incorrect template | Check margin settings |
| Jamming | Dirty rollers | Clean with alcohol swab |

#### Scanning Issues
| Issue | Likely Cause | Solution |
|-------|-------------|----------|
| Won't scan | Low contrast | Adjust barcode darkness |
| Wrong attendee | Data mismatch | Re-export attendee list |
| Multiple scans | Barcode too small | Increase size to 1.5" width |
