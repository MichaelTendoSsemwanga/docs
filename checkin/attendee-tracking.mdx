# Complete Attendee Management Guide

*Streamline your attendee experience with Loft's powerful management tools. This guide covers everything from registration to post-event follow-up, ensuring you can manage attendees efficiently at any event scale.*

## Quick Start: Manage Attendees

### Step 1: Access Attendee Dashboard
1. Log in to [Loft Dashboard](https://app.loft.events)
2. Select your event
3. Navigate to **Attendees** > **All Attendees**

### Step 2: Key Actions
- **Add Attendees**: Manually or import CSV
- **Check-in**: Scan tickets or search by name
- **Filter/Search**: Find specific attendees
- **Export**: Download attendee data

## Core Attendee Management

### 1. Adding & Importing Attendees

#### A. Single Attendee
1. Click **+ Add Attendee**
2. Fill in details:
   - Required: Name, Email
   - Optional: Company, Job Title, Custom Fields
3. Assign ticket type
4. Send confirmation (optional)

#### B. Bulk Import
1. **Prepare CSV File**
   - Download template
   - Required columns: `first_name`, `last_name`, `email`
   - Add custom fields as needed

2. **Upload File**
   - Go to **Attendees** > **Import**
   - Map CSV columns to fields
   - Preview & confirm
   - Set email preferences

### 2. Check-in Management

#### A. Check-in Stations
1. **Set Up**
   - Install Loft Check-in App
   - Log in with organizer credentials
   - Download event data

2. **Check-in Methods**
   - **QR Code Scan**: Attendee shows ticket on phone
   - **Name Search**: Type 3+ letters of name
   - **Manual Check-in**: No ticket required

#### B. Check-in Workflows
1. **Standard Check-in**
   - Scan ticket
   - Verify photo ID (if enabled)
   - Print badge (if configured)
   - Hand out materials

2. **On-site Registration**
   - Select "New Registration"
   - Enter attendee details
   - Process payment (if applicable)
   - Complete check-in

### 3. Attendee Communication

#### A. Email Campaigns
1. **Create Email**
   - Go to **Email** > **New Campaign**
   - Choose template
   - Personalize with merge tags
   - Schedule or send immediately

2. **Segmentation**
   - Filter by:
     - Ticket type
     - Check-in status
     - Session attendance
     - Custom fields

#### B. SMS Notifications
1. **Send Alerts**
   - Select "SMS" in Comms
   - Choose recipients
   - Type message (160 char limit)
   - Send test first

## Advanced Features

### 1. Badge Printing
1. **Design Badges**
   - Go to **Check-in** > **Badge Designer**
   - Drag & drop fields
   - Add logo and branding
   - Set print layout

2. **Print Options**
   - Single badge

### 2. Session Tracking
1. **Track Attendance**
   - Enable session scanning

2. **Engagement Reports**
   - Session popularity

   - Dwell time analysis

## Step 1: Set Up Check-in Stations

### 1.1 Hardware Setup
- **Devices Needed**:
  - 1 tablet/smartphone per 100 attendees
  - Sturdy stands or mounts
  - Power banks (2 per device)
  - Backup paper lists (10% of total attendees)

### 1.2 Install Check-in App
1. **For iOS**:
   - Open App Store
   - Search "Loft Check-in"
   - Tap **Get** and install

2. **For Android**:
   - Open Google Play Store
   - Search "Loft Check-in"
   - Tap **Install**

### 1.3 Configure Each Device
1. Open the Loft Check-in App
2. Log in with your organizer account
3. Tap **Select Event**
4. Enable **Offline Mode**
5. Tap **Download Attendee List**
6. Wait for sync to complete

## Step 2: Import & Manage Attendee Lists

### 2.1 Prepare Your Attendee List
- **Required Fields**:
  - First Name
  - Last Name
  - Email
  - Ticket Type
  - Order Number

- **Download Template**:
  1. Go to your Loft dashboard
  2. Navigate to **Attendees** > **Import**
  3. Click **Download Template**

### 2.2 Import Attendees
1. Go to **Attendees** > **Import**
2. Click **Choose File**
3. Select your CSV/Excel file
4. Map columns to fields
5. Click **Import**
6. Review and confirm

### 2.3 Update Attendee Information
1. **Single Edit**:
   - Find attendee
   - Click **Edit**
   - Make changes
   - Click **Save**

2. **Bulk Edit**:
   - Select multiple attendees
   - Click **Bulk Actions**
   - Choose field to update
   - Enter new value
   - Click **Apply**

## Step 3: Check-in Attendees

### 3.1 Standard Check-in
1. Open the Check-in App
2. Tap **Check In**
3. Scan attendee's QR code
4. Verify details
5. Tap **Check In**

### 3.2 Manual Check-in
1. Tap **Search** (ðŸ”)
2. Enter attendee's:
   - Name
   - Email
   - Last 4 digits of order number
3. Select from results
4. Tap **Check In**

### 3.3 Group Check-in
1. Tap **Group Check-in**
2. Scan all tickets in the group
3. Review the list
4. Tap **Check In All**

## Step 4: Track Attendance in Real-time

### 4.1 View Live Dashboard
1. Go to **Dashboard**
2. View key metrics:
   - Total Checked In
   - Check-in Rate
   - Peak Times
   - Staff Performance

### 4.2 Set Up Alerts
1. Go to **Settings** > **Alerts**
2. Enable:
   - VIP Arrivals
   - Duplicate Check-ins
   - Capacity Warnings
3. Choose notification method:
   - Push Notifications
   - Email Alerts
   - SMS (premium)

## Step 5: Generate Reports

### 5.1 Standard Reports
1. Go to **Reports**
2. Select report type:
   - **Check-in Summary**: Overall attendance
   - **No-Shows**: Who didn't attend
   - **Peak Times**: Busiest periods
   - **Staff Performance**: Check-ins per staff
3. Set date range
4. Click **Generate**

### 5.2 Custom Reports
1. Click **Create Custom Report**
2. Select data points:
   - Attendee Information
   - Check-in Times
   - Session Attendance
   - Custom Fields
3. Add filters
4. Choose format (CSV/Excel/PDF)
5. Click **Generate**

## Pro Tips for Large Events

### Before the Event
- **Prepare Multiple Stations**:
  - 1 station per 100 attendees
  - Label each station (A, B, C...)
  - Assign staff to specific stations

- **Test Everything**:
  - Do a full dress rehearsal
  - Test with 10% of attendee list
  - Verify all devices sync properly

### During the Event
- **Manage Queues**:
  - Use stanchions for lines
  - Post clear signage
  - Have staff direct traffic

- **Staff Communication**:
  - Use walkie-talkies
  - Designate a command center
  - Schedule breaks for staff

### After the Event
- **Data Backup**:
  - Export all check-in data
  - Save multiple copies
  - Store in cloud storage

- **Follow-up**:
  - Send thank you emails
  - Share event photos
  - Request feedback

## Troubleshooting

### Common Issues & Solutions

#### "Attendee Not Found"
1. Verify spelling
2. Check registration status
3. Look for typos in email
4. Try searching by order number

#### Duplicate Check-ins
1. Check check-in history
2. Look for similar names
3. Verify email addresses
4. Merge duplicate records if needed

#### System Offline
1. Check Wi-Fi/Data connection
2. Toggle Airplane mode
3. Switch to Offline Mode
4. Continue checking in - data will sync later
