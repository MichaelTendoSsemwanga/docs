# Theme Customization Guide

Customize your help center's appearance to match your brand identity with our comprehensive theming options.

## Brand Colors

### Primary Color
Your primary brand color is used for buttons, links, and other interactive elements.

```css
:root {
  --primary-50: #f0f9ff;
  --primary-100: #e0f2fe;
  --primary-200: #bae6fd;
  --primary-300: #7dd3fc;
  --primary-400: #38bdf8;
  --primary-500: #0ea5e9;
  --primary-600: #0284c7;
  --primary-700: #0369a1;
  --primary-800: #075985;
  --primary-900: #0c4a6e;
}
```

### Customizing Colors
1. Go to **Settings** > **Appearance** > **Colors**
2. Use the color picker or enter hex codes
3. Preview changes in real-time
4. Click **Save** to apply

## Typography

### Font Family
Choose from our curated selection of fonts or upload your own.

#### Recommended Fonts
- **Sans-serif**: Inter, Open Sans, Roboto
- **Serif**: Merriweather, Lora, Playfair Display
- **Monospace**: Fira Code, Roboto Mono

### Font Sizes
Customize the typography scale for better readability.

```css
:root {
  --font-size-xs: 0.75rem;   /* 12px */
  --font-size-sm: 0.875rem;  /* 14px */
  --font-size-base: 1rem;    /* 16px */
  --font-size-lg: 1.125rem;  /* 18px */
  --font-size-xl: 1.25rem;   /* 20px */
  --font-size-2xl: 1.5rem;   /* 24px */
  --font-size-3xl: 1.875rem; /* 30px */
}
```

## Layout

### Header & Navigation
- **Sticky Header**: Enable/disable
- **Navigation Style**: Sidebar or Topbar
- **Search Position**: Header or Hero section

### Content Width
- **Narrow**: 800px (better for reading)
- **Medium**: 1000px (balanced)
- **Wide**: 1200px (for rich content)
- **Full Width**: 100% of the container

## Custom CSS/JS

### Adding Custom CSS
1. Go to **Settings** > **Advanced** > **Custom CSS**
2. Enter your CSS in the editor
3. Use the preview to see changes
4. Click **Save & Publish**

### Example Custom CSS
```css
/* Custom button styles */
.btn-primary {
  @apply bg-indigo-600 hover:bg-indigo-700 text-white font-medium rounded-lg px-6 py-2.5;
  transition: all 0.2s ease-in-out;
}

/* Custom card styles */
.help-card {
  @apply bg-white rounded-xl shadow-sm border border-gray-200 p-6;
  transition: transform 0.2s ease-in-out, box-shadow 0.2s ease-in-out;
}

.help-card:hover {
  @apply shadow-md;
  transform: translateY(-2px);
}
```

Choose from our pre-designed themes or create your own.

3. **Bold** - High contrast with vibrant colors

## Preview & Publish

### Preview Mode
- Test your changes before going live
- Share preview links with your team
- Schedule publishing for a specific time

### Version History
- Track all theme changes
- Roll back to previous versions
- Compare changes side by side

## Best Practices

### Accessibility
- Ensure sufficient color contrast
- Use semantic HTML elements
- Test keyboard navigation
- Add proper ARIA labels

### Performance
- Optimize images before uploading
- Minimize custom CSS/JS
- Use web-safe fonts or preload custom fonts
- Enable compression and caching
