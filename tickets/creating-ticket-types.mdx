# Creating & Managing Ticket Types

*Set up the perfect ticket strategy for your event with Loft's flexible ticketing system. This guide walks you through creating, customizing, and optimizing ticket types to maximize attendance and revenue.*

## Quick Start: Create Your First Ticket Type

### Step 1: Access Ticket Management
1. Log in to [Loft Dashboard](https://app.loft.events)
2. Select your event or create a new one
3. Navigate to **Tickets** > **Manage Ticket Types**
4. Click **+ Create Ticket Type**

### Step 2: Configure Basic Settings
1. **Ticket Name & Description**
   - Use clear, descriptive names (e.g., "VIP All-Access Pass - Includes After Party")
   - Add benefits in the description (limit 200 characters)
   - Example: `##VIP Access##\n• Premium seating\n• Exclusive lounge access\n• Meet & greet with speakers`

2. **Pricing & Availability**
   - Set base price (exclude fees)
   - Choose currency (if multiple supported)
   - Set sales start/end dates
   - Enable/disable after sale ends

3. **Inventory & Limits**
   - Set total quantity available
   - Enable waitlist when sold out
   - Set purchase limits per order (min/max)

## Common Ticket Types & Setups

### 1. General Admission
**Use Case**: Standard event access  
**Setup**:
1. Name: "General Admission"
2. Price: $99
3. Quantity: 500
4. Sales: Event start to end date

### 2. Early Bird Pricing
**Use Case**: Limited-time discount  
**Setup**:
1. Name: "Early Bird (Save 30%)"
2. Price: $69 (regular $99)
3. Quantity: 100
4. Sales: Today until [2 weeks before event]

### 3. VIP/All-Access
**Use Case**: Premium experience  
**Setup**:
1. Name: "VIP Experience"
2. Price: $299
3. Benefits field: List VIP perks
4. Limit: 50 tickets

### 4. Group Tickets
**Use Case**: Bulk discounts  
**Setup**:
1. Name: "Group of 5 (Save 20%)"
2. Price: $400 ($80/person)
3. Min/Max: 5 tickets per order
4. Add note: "Must purchase 5 tickets"

## Creating Ticket Types

### Step 1: Access Ticket Management
1. Log in to your Loft account
2. Select your event from the dashboard
3. Click **Tickets** in the left navigation menu
4. Click **+ Add Ticket Type**

### Step 2: Configure Basic Information

#### Required Fields
- **Ticket Name** (Max 50 characters)
  - Be specific and descriptive
  - Example: `VIP All-Access Pass` or `Student General Admission`
  
- **Description** (Supports Markdown)
  - Clearly state what's included
  - List benefits or restrictions
  - Example:
    ```
    ## VIP Benefits
    - Priority seating
    - Exclusive lounge access
    - Meet & greet with speakers
    ```

- **Price**
  - Set base price (or mark as free)
  - Add fees (flat or percentage)
  - Choose if fees are absorbed or passed to buyer

#### Optional Fields
- **SKU/Code**: For inventory tracking
- **Sales Start/End**: Automate ticket availability
- **Quantity Available**: Set maximum capacity

### Step 3: Configure Access & Restrictions

#### Visibility Settings
| Option | Description | Use Case |
|--------|-------------|----------|
| **Public** | Visible to all | General admission |
| **Password Protected** | Requires access code | Private events |
| **Hidden** | Direct link only | Special guests |
| **Members Only** | For logged-in members | Member exclusives |

#### Purchase Limits
- **Per Order**: Min/Max tickets per transaction
- **Per Customer**: Total tickets per person/email
- **Custom Rules**: Based on attendee attributes
