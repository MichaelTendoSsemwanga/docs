# Discounts & Promotions

Create and manage discounts and promotional offers to boost ticket sales for your events.

## Creating Discount Codes

### Basic Setup
1. Navigate to your event dashboard
2. Go to **Tickets** > **Discounts**
3. Click **Create Discount Code**
4. Enter the following details:
   - **Code**: The promotional code attendees will use
   - **Discount Type**: Percentage or fixed amount
   - **Value**: The discount amount
   - **Usage Limit**: Maximum number of times the code can be used
   - **Valid From/To**: Date range for the discount

### Advanced Options
- **Minimum Order Value**: Set a minimum cart total required
- **Applies To**: Specific ticket types or all tickets
- **One-time Use**: Limit to one use per customer
- **Hidden**: Create private codes for specific audiences

## Promotion Types

### Early Bird Discounts
- Set time-limited discounts to encourage early sales
- Automatically switch to regular pricing after the deadline

### Group Discounts
- Offer discounts for bulk purchases
- Set tiered pricing (e.g., 10% off for 5+ tickets, 15% off for 10+)

### Member/Student Discounts
- Create special pricing for specific groups
- Require verification if needed

### Promotional Pricing
- Limited-time price reductions
- Flash sales with countdown timers

## Managing Promotions

### Monitoring Performance
- Track redemption rates
- Monitor revenue impact
- View which ticket types are most popular with discounts

### Pausing/Ending Promotions
- Temporarily disable discount codes
- Set automatic expiration dates
- Extend successful promotions

## Best Practices

### Code Naming
- Use all caps for better readability
- Keep codes short but memorable
- Avoid easily confused characters (0/O, 1/I/l)

### Promotion Strategy
- Test different discount amounts
- Create urgency with time limits
- Target specific customer segments
- Analyze which promotions drive the most sales

## Troubleshooting

### Common Issues
- **Code not working**: Check validity dates and usage limits
- **Discount not applying**: Verify ticket type eligibility
- **Multiple discounts**: Only one code can be used per order
