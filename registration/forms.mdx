# Registration Forms with Loft

*Create seamless registration experiences with Loft's powerful form builder. This guide covers everything from basic setup to advanced customizations for your event registration forms.*

## Quick Start

### Accessing Form Builder
1. Log in to your Loft account
2. Go to **Events** > **Your Event** > **Registration**
3. Click **Form Builder**

### Key Features
- Drag-and-drop form builder
- Custom field types
- Conditional logic
- Multi-step forms
- File uploads
- Payment integration

## Step 1: Create a New Form

### 1.1 Basic Setup
1. Click **+ New Form**
2. Enter form name (e.g., "Attendee Registration")
3. Select form type:
   - **Attendee Registration**
   - **Speaker Application**
   - **Sponsor Inquiry**
   - **Custom Form**

### 1.2 Form Settings
1. **General**
   - Form title and description
   - Submission button text
   - Confirmation message

2. **Access Control**
   - Public/Private access
   - Password protection
   - Registration deadlines

## Step 2: Add Form Fields

### 2.1 Standard Fields
1. **Personal Information**
   - Name (First/Last)
   - Email
   - Phone
   - Company
   - Job Title

2. **Event-Specific Fields**
   - Session preferences
   - Dietary restrictions
   - Accessibility needs
   - T-shirt size

### 2.2 Advanced Fields
1. **File Uploads**
   - Resume/CV
   - Presentation slides
   - Photo

2. **Payment Fields**
   - Ticket selection
   - Donation options
   - Coupon codes

## Step 3: Configure Field Settings

### 3.1 Field Properties
1. **Basic Settings**
   - Field label
   - Placeholder text
   - Help text
   - Required/Optional

2. **Validation Rules**
   - Email format
   - Phone number format
   - Character limits
   - File type restrictions

### 3.2 Conditional Logic
1. **Show/Hide Fields**
   - Based on previous answers
   - User type
   - Ticket selection

2. **Skip Logic**
   - Skip to specific sections
   - End form early if criteria met

## Step 4: Form Design

### 4.1 Styling Options
1. **Themes**
   - Pre-built templates
   - Custom color schemes
   - Font selection

2. **Layout**
   - One-page vs. multi-step
   - Section breaks
   - Progress indicator

### 4.2 Branding
1. **Logo & Images**
   - Upload logo (PNG, 200x50px)
   - Header/footer images
   - Custom favicon

2. **Custom CSS**
   - Advanced styling
   - Responsive design
   - Custom classes

## Step 5: Form Actions

### 5.1 Confirmation
1. **Success Message**
   - Custom thank you text
   - Next steps
   - Add to calendar

2. **Notifications**
   - Admin notifications
   - Auto-responders
   - Custom email templates

### 5.2 Integrations
1. **CRM**
   - Salesforce
   - HubSpot
   - Pipedrive

2. **Marketing Tools**
   - Mailchimp
   - ActiveCampaign
   - ConvertKit

## Step 6: Publish & Share

### 6.1 Publish Options
1. **Embed Code**
   - Copy/paste into website
   - Responsive iframe
   - Popup window

2. **Direct Link**
   - Shareable URL
   - QR code
   - Short link

### 6.2 Access Control
1. **Restrictions**
   - Password protection
   - Email domain whitelist
   - Maximum submissions

2. **Scheduling**
   - Open/close dates
   - Timezone settings
   - Limited time offers

## Troubleshooting

### Common Issues

#### Form Not Submitting
1. Check required fields
2. Verify file sizes
3. Test payment gateway

#### Conditional Logic Not Working
1. Check rule conditions
2. Verify field IDs
3. Test all scenarios
