# Event Reporting & Analytics Guide

*Unlock the power of your event data with Loft's comprehensive analytics. This guide helps you measure success, understand your audience, and make data-driven decisions.*

## Quick Start: Generate Your First Report

### Step 1: Access Analytics Dashboard
1. Log in to [Loft Dashboard](https://app.loft.events)
2. Select your event
3. Navigate to **Analytics** > **Reports**
4. Choose a template or start from scratch

### Step 2: Run & Customize
1. Select date range
2. Choose metrics (registrations, revenue, etc.)
3. Add filters (ticket type, source, etc.)
4. Click **Generate Report**

## Advanced Reporting Features

### 1. Custom Report Builder
1. **Create Custom Reports**
   - Go to **Analytics** > **Custom Reports**
   - Drag & drop metrics
   - Add custom calculations
   - Save as template

2. **Scheduled Reports**
   - Set up automatic delivery
   - Choose frequency (daily/weekly/monthly)
   - Select recipients
   - Set expiration date

### 2. Data Export & Integration
1. **Export Options**
   - CSV/Excel/PDF formats
   - Custom column selection
   - Raw data vs. aggregated

2. **Third-Party Integration**
   - Connect to Google Analytics
   - Push to Google Sheets
   - API access for custom solutions
